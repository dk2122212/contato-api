# Atividade 3: Spring Boot – API Contato (v1)

Este projeto é uma API REST simples desenvolvida para a disciplina de **Back-end Frameworks** da UNINASSAU.

## 🚀 Objetivo

Criação da estrutura inicial de uma API de Contatos utilizando Spring Boot, com os quatro métodos HTTP principais (GET, POST, PUT, DELETE).

## 🛠️ Tecnologias

- **Java**
- **Spring Boot**
- **Maven**

## 📌 Endpoints

- **GET** `/contatos` - Lista contatos.
- **POST** `/contatos` - Cria contato.
- **PUT** `/contatos` - Atualiza contato.
- **DELETE** `/contatos` - Remove contato.
